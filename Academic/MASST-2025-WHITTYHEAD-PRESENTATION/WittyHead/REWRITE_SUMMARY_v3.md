## Key Innovations Highlighted

### 1. Empathetic (Non-Mirroring) Response Mapping

**Table 1**: Shows asymmetric mappings
- User Sad → Avatar Compassionate Concern (NOT mirrored sadness)
- User Angry → Avatar Calm Concern (de-escalation)
- User Fear → Avatar Reassuring
- User Happy → Avatar Happy (mirror positive emotions)

**Scientific Basis**: Gilbert et al. (2019) research showing mirrored distress is "invalidating and aversive"

### 2. Multi-Modal Coordination

- **Facial** (ARKit/FACS): AU4 + AU6 + gentle AU12 for compassionate concern
- **Gaze** (60-90% eye contact): Therapeutic alliance research
- **Gesture** (prosodic-aligned): Loehr (2012) synchrony
- **Voice** (prosody): Scherer (2003) vocal emotion

### 3. Supporting Vulnerable Populations

- **Social isolation**: Foster care youth, elderly, refugees
- **Mental health**: Depression, anxiety, trauma support
- **Disability**: Visual, auditory, motor, cognitive accommodation
- **Cultural sensitivity**: Easy Japanese, cultural ontologies

### 4. Digital MOAI Integration

- **Privacy-preserving**: Local-first emotion processing
- **User-controlled**: Three automation levels
- **Group-aware**: MOAI collective context
- **Emergency support**: Coordinated distress response
