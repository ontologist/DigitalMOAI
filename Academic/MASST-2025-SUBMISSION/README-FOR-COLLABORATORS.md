## üíª Option 2: Edit Locally with LaTeX

**Best for:** Offline editing, experienced LaTeX users

### Prerequisites:
- LaTeX distribution installed:
  - **Mac:** MacTeX (https://tug.org/mactex/)
  - **Windows:** MiKTeX (https://miktex.org/)
  - **Linux:** TeX Live (usually pre-installed)

### Compilation Steps:

```bash
# Navigate to directory with files
cd /path/to/MASST-2025-files

# Compile (run these commands in order):
pdflatex MASST-2025-Abstract.tex
bibtex MASST-2025-Abstract
pdflatex MASST-2025-Abstract.tex
pdflatex MASST-2025-Abstract.tex

# Output: MASST-2025-Abstract.pdf
```

### Why 4 compilation steps?
1. First `pdflatex`: Generate auxiliary files
2. `bibtex`: Process bibliography
3. Second `pdflatex`: Include bibliography
4. Third `pdflatex`: Resolve all cross-references
